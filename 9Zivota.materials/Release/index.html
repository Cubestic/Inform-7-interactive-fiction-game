A work of interactive fiction.

9Zivota was created with Inform and has IFID CAD75681-8B0E-4B24-A7CB-B9DE639CDA8D. To play a work like this one, you need an interpreter program: many are available, among them Gargoyle for MacOS, Linux or Windows; Spatterlight for MacOS; Windows Frotz or Windows Glulxe for Windows. Play without downloading anything by following the 'Play In-Browser' link, using the Vorple interpreter. You'll need to have Javascript enabled on your web browser.
